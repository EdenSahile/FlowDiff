## Prochaine feature à implémenter : Références de commande sur les ORDERS EDI

### Contexte métier
Quand un libraire passe une commande, il doit pouvoir y associer une **référence** (ex : numéro de bon de commande interne, nom d'un projet, etc.) pour la retrouver facilement dans son système de gestion. Cette référence doit :
- Être incluse dans le message EDIFACT ORDERS (segment `RFF+CR:<ref>'` — Customer Reference)
- Être visible dans le détail de la commande (historique, modal EDI)

### Deux modes au choix (à proposer dans le panier)
1. **Référence globale** — une seule référence pour toute la commande (s'applique à toutes les lignes)
2. **Référence par ligne** — une référence différente pour chaque ouvrage (ex : affectation par rayon, par client)

### Fichiers à modifier

#### 1. Types et données
- `src/data/mockOrders.ts` — ajouter `referenceCommande?: string` sur `Order` et `referenceCmd?: string` sur `OrderItem`
- `src/lib/ediUtils.ts` — ajouter `referenceCommande?: string` et `referenceLigne?: string` sur `ORDERSLine` ; dans `ORDERSPayload` ajouter `referenceGlobale?: string`

#### 2. Panier (saisie de la référence)
- `src/pages/cart/CartPage.tsx` — ajouter :
  - Un champ texte "Référence commande" (optionnel, placé dans le récapitulatif en haut ou avant le bouton "Valider")
  - Un toggle ou radio "Référence globale / Par article"
  - Si "Par article" : afficher un input de référence sous chaque ligne du panier
- `src/contexts/CartContext.tsx` (ou `OrdersContext.tsx`) — stocker la/les références lors de la validation

#### 3. Template EDIFACT ORDERS
- `src/lib/ediUtils.ts` — dans le template `ORDERS` :
  - Si référence globale : ajouter `RFF+CR:<referenceGlobale>'` après les NAD (avant les lignes)
  - Si référence par ligne : ajouter `RFF+CR:<referenceLigne>'` après chaque `QTY+21` dans les lignes

#### 4. Affichage dans l'historique
- `src/pages/historique/HistoriquePage.tsx` (ou composant de détail commande) — afficher la référence si présente

### Segment EDIFACT cible
```
RFF+CR:REF-INTERNE-001'   ← référence globale (après NAD)
...ou par ligne :
LIN+1++9782070360024:EN'
IMD+F++:::Le Voyageur des confins T.1'
QTY+21:3'
RFF+CR:RAYON-LITTERATURE'  ← référence ligne
```
