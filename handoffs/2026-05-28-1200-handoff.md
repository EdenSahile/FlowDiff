## ⚠️ Bug critique en production — NON corrigé

**Symptôme :** pages blanches sur Fonds, Nouveautés, À paraître, Top Ventes, Sélections

**Cause racine identifiée :** cinq fichiers référencent des IDs de livres supprimés de `mockBooks.ts` :

| Fichier | Problème | IDs orphelins (exemples) |
|---------|----------|--------------------------|
| `src/data/mockSeries.ts` | `coverBookId` + `bookIds` | `f-bd-01`, `f-lit-01`, `s-ttn-01`, `f-jes-01`… |
| `src/data/mockSelections.ts` | `bookIds` dans chaque sélection | `f-bd-01`, `f-lit-02`, `f-jes-01`… |
| `src/pages/top-ventes/TopVentesPage.tsx` | objet `SALES` hardcodé ligne 32–41 | `f-bd-01`, `f-bd-02`, `f-lit-01`… |
| `src/data/mockFlashInfos.ts` | `bookId` dans les flash infos | `f-jes-01`, `f-jes-02`, `f-lit-01`, `f-pra-01` |
| `src/data/mockOrders.ts` | `bookId` dans les lignes de commande | `f-lit-01`, `f-bd-01`, `f-jes-01`… |

Quand ces composants appellent `getBookById(id)` avec un ID inexistant → `undefined` → lecture de `.isbn` / `.title` → crash → page blanche.
