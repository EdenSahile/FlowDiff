# FlowDiff PRO

Application B2B de commande de livres à destination exclusive des libraires.

---

## Stack

- **Vite 5 + React 18** — SPA, routing React Router v6
- **TypeScript** (strict)
- **Styled-components v6**
- **Vitest** — tests unitaires

## Lancer en local

```bash
npm install
npm run dev
```

L'app tourne sur [http://localhost:5173](http://localhost:5173).

## Tests

```bash
npm run test
```

## Déploiement

Le projet est déployé sur **Vercel** (build Vite automatique sur push `main`).
